# Vs (UMAP & t-SNE)

## Description

| Feature                   | UMAP                     | t-SNE                        |
| ------------------------- | ------------------------ | ---------------------------- |
| **Speed**                 | Faster                   | Slower                       |
| **Scalability**           | Handles large data well  | Struggles with large data    |
| **Global Structure**      | Preserves better         | Often loses global structure |
| **Local Structure**       | Very good                | Excellent                    |
| **Parameters to Tune**    | Fewer                    | More                         |
| **Embedding Consistency** | More stable across runs  | Less stable                  |
| **Dimensionality Output** | Any number of dimensions | Typically 2 or 3 only        |
